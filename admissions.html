Schedule Inpatient Episode and Admit Patient Efficiently Using Medworkflow

This tutorial explains how to schedule an inpatient episode when the patient is ready for admission but not yet scheduled by the doctor's rooms. You will learn the complete process from patient search to final admission confirmation.

Go to medworkflow.co.za

1. Overview of Scheduling Inpatient Episode

Today I will be demonstrating how to schedule an inpatient episode if the doctor's rooms did not schedule the patient but the patient is at the hospital ready to be admitted.

2. Search Patient and Schedule Option

So the first step you will see is once you search for the patient you will not find any results. Then it will display an option to schedule a new episode.

3. Enter Patient ID for Verification

We start by entering the patient ID number. This will check if the patient has been at the hospital before or if we have their data. It will display "patient found" if that is the case, and you can simply continue.

4. View Patient Admission Data

Here it will display the patient information that has been pulled automatically from the databases for the first step on admission information.

5. Manually Enter Admission Data

If the patient was not on the databases, you will need to fill out this step. Additionally, you can fill out what the admission type is, if it's a surgical case or a medical case.

6. Set Admission Type and Authorization

Furthermore, you can add the admission date, and if post-op eye care is required, you can toggle it and add the authorization number.

7. Provide Authorization Number

An authorization number is required to admit the patient. Once you save and continue on this step, it will automatically send a pre-admission link to the patient where they can confirm all of their data if it is on the system already, or enter all of their data if we do not have it yet.

8. Save and Continue Admission

Click Save and Continue to proceed with the admission process.

9. Select Primary Admitting Doctor

The next step will be to select the doctors assigned to this case. First of all, the primary doctor or admitting doctor.

10. Choose Referring or Family Doctor

This is the surgeon who will be operating on the patient. Next, the referring or family doctor. Either referring or family doctor is required before admission.

11. Search and Select Doctor

Admission, so you can simply search and select the applicable doctor if

12. Click to Proceed

Click here to continue to the next step.

13. Add New Doctor Function

The doctor is not available. You can simply select the Add New Doctor function here.

14. Enter Doctor's First Name

Click the first name field to enter the new doctor's first name.

15. Input New Doctor Details

You can enter the doctor name, last name, specialty, and practice area. This will add a new doctor.

16. Confirm Doctor Selection

If there are similar doctors, it might ask you if this is the doctor you are looking for. If you didn't find the doctor in the search, you can simply click on "Use this doctor." If this is not the doctor you were looking for and you need to add a new doctor, you can simply click "Add new."

17. Save and Continue Doctor Selection

Click Save and Continue to proceed after selecting the doctor.

18. Enter Coding and Procedure Info

On this page, we need to enter all the coding information and procedure information, ICD-10 codes. I am simply taking anything at this point just for demonstration to automatically fill in the description field, RPL code.

19. Note on RPL Code Format

Note an RPL code is always a four digit code and it will not accept a five digit code if you try to continue.

20. Access ICD-10 Code Entry

Click the ICD-10 code field to enter the primary diagnosis code.

21. Enter Surgical Procedure Description

Click the surgical procedure description field to provide details about the procedure.

22. Describe Surgical Procedure

Surgical procedure description.

23. Enter Procedure Description

Enter your surgical procedure description for demonstration purposes.

24. Complete Required Fields

The required fields indicated by a red asterisk are completed. You can continue.

25. Enter Medical Aid Information

Here you will be required to enter the patient medical aid information. Please note that the patient will also enter this on the pre-admission.

26. Save and Continue Medical Aid

Click Save and Continue to proceed after entering medical aid information.

27. Book Theatre Slot Overview

Finally, you will need to book a theatre slot for the patient. We will select the theatre date first.

28. Select Theatre Date and Duration

Once the date is selected, you can enter the estimated duration of the procedure and the requested start time.

29. Add Additional Theatre Notes

Click the additional notes field to request a theatre and time for approval.

30. Automatic Theatre Slot Assignment

Note that on this step it will automatically select an assigned theatre slot if the doctor has an assigned theatre slot for the date.

31. Manual Theatre Booking Approval

If there is no assigned theatre slot, it will use the manual approved booking option. That simply means that the case will be submitted to theatre management for approval.

32. Continue to Next Step

Click Continue to proceed with the booking process.

33. Upload Surgery Consent and Card

Here you can upload the surgery consent if you have it on hand, medical aid card. This will be uploaded by the patient as well.

34. Save Booking and Pre-Admission

You can save the case, then the booking is complete. During this time, the patient will have completed their pre-admission because it was sent at the admission information.

35. Review Doctor Information

Click Review Doctor Information to verify the assigned doctors.

36. Wait for Pre-Admission Completion

At this point you can wait for the patient to complete the pre-admission before continuing. Once the patient has completed the pre-admission you can continue.

37. Review Patient Pre-Admission Data

You can review all of the information, the guarantor information, the medical aid information, the next of kin, the clinical information. Once again, this is all information that has been entered by the patient on their pre-admission.

38. Request and Upload Documents

Then you can review that the patient uploaded all of their documents. As you can see here, the documents have not been uploaded. So you will need to request the patient ID document, scan and upload it. Same with the medical aid card. And if any of the other applicable documents are available, you can upload those as well.
