Theatre Operations Training Guide

Add and Schedule Surgical Cases

A structured guide for patient search, clinical case capture, doctor assignment and theatre schedule management.

46 guided stepsTheatre managementDigital walkthrough included

Interactive walkthrough

This tutorial guides you through adding and scheduling surgical cases in Medworkflow.

Go to sandbox.medworkflow.co.za

1. Introduction

You will learn how to search for patients, assign doctors, and manage theatre schedules effectively.

2. Initiate New Case Entry

Click "Add case" to start creating a new patient case in Medworkflow.

3. Access Patient ID Field

Click "SA ID or passport number" to specify the patient's identification for search.

4. Enter Patient ID Number

Enter the patient's identification number to locate their records in the system.

5. Search For Patient

Click "Search Patient" to find the patient using the entered identification number.

6. Proceed After Patient Search

Click "Continue" to move forward after confirming the patient details.

7. Select Case Type Category

complete the required fields with the patient's details.

8. Choose Surgical Case Type

Click "Surgical" to specify that the case involves a surgical procedure.

9. Access Date Selection

Click here to open the date selection for the surgical case.

10. Select Surgery Date

11. Save Date And Continue

Click "Save and Continue" to confirm the selected date and proceed.

12. Open Doctor Selection

Click here to access the list of available doctors for assignment.

13. Add New Doctor

14. Assign Referring Doctor

15. Save Doctor Assignment

Click "Save and Continue" to confirm the doctor assignment and proceed.

16. Select Diagnosis Code

choose the correct diagnosis codes for the case

17. Confirm Diagnosis Code

18. Add Additional ICD-10 Code

19. Select Additional Diagnosis

20. Access Procedure Code Field

Click here to open the procedure code input area.

21. Select Procedure Code

22. Enter Procedure Code

Enter the procedure code number to identify the surgical procedure.

23. Open Procedure Description

Click " Surgical Procedure Description." to add details about the surgery.

24. Describe Surgical Procedure

Enter a brief description of the surgical procedure to provide necessary details.

25. Save Procedure Details

Click "Save and Continue" to confirm the procedure information and proceed.

26. Open Time Selection

Click here to access the time selection for scheduling the surgery.

27. Select Surgery Time

28. Confirm Time Selection

Click "Confirm" to finalize the selected time for the surgery.

29. Enter Estimated Duration

Click "Enter Estimated Minutes" to specify the expected length of the surgery.

30. Fill Surgery Duration

indicate the estimated duration of the surgical procedure in minutes.

31. Access Theatre Selection

Click here to open the theatre selection menu.

32. Choose Theatre Room

select the theatre where the surgery will take place.

33. Select Start Time

set the surgery start time.

34. Select End Time

35. Proceed After Scheduling

Click "Continue" to move forward after scheduling the surgery.

36. Assign Doctor To Case

To add unconfirmed cases to the list, drag the case from the left panel and drop it into the desired theater.

37. Select Theatre 1

To allocate a new theatre slot to a doctor Click "Here"

38. Select Theatre 3

Choose the theatre

39. Open Recurrence Options

Select the day you prefer for the slot.

40. Select Weekly Recurrence

41. Confirm Weekly Recurrence

Click "Here" to confirm the weekly recurrence setting.

42. Open Doctor Selection

Click "Select doctor" to choose a doctor for the recurring schedule.

43. Choose Doctor For Recurrence

44. Assign Doctor To Schedule

Click "Assign" to finalize the doctor assignment for the recurring slot.

45. Clear Scheduled Day

Click "Clear Day" to remove the slot.

46. Approve And Send List

When you are done creating your list Click "Approve and Send Theatre List" to finalize and distribute the theatre schedule.
You have successfully added and scheduled a surgical case in Medworkflow, including patient search, doctor assignment, procedure coding, and theatre scheduling. Next, you can manage recurring surgeries or review theatre lists for efficient operation management.
