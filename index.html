Schedule and Manage Patient Episodes

A structured, step-by-step guide for capturing patient information, assigning clinical details and completing a patient episode accurately.

51 guided stepsPatient administrationDigital walkthrough included

Interactive walkthrough

This tutorial guides you through scheduling a new patient episode in Medworkflow.

Go to medworkflow.co.za

1. Introduction

You will learn how to enter patient details, select medical options, and finalize the case efficiently.

2. Initiate New Episode Scheduling

Click "Schedule new Episode" to start creating a new patient episode in Medworkflow.

3. Access Patient ID Entry

Click "Enter patient ID number" to input the patient's identification details.

4. Enter Patient ID Number

Enter the patient's ID number in the field provided.

5. Proceed After ID Entry

Click "Continue" to move to the next step after entering the patient ID.

6. Open ID Type Selection

Click "Select ID type" to choose the type of identification provided.

7. Choose Passport Number Type

Select the appropriate ID type for your patient: use a South African ID for South African citizens and a passport number for those from abroad.

8. Confirm ID Type Selection

Click "Continue" to confirm the selected ID type and proceed.

9. fill in all the required fields under patient information

Complete all required fields in the patient information section.

10. Select Medical or Surgical Category

Click "Admission type" to specify the nature of the patient's case.

11. Choose Surgical Category

12. Select Private Option

Click "Patient type" to indicate the patient's payment or medical aid status.

13. Choose Medical Aid Option

14. Access Authorization Number Field

Click "Authorization Number" to enter the required authorization details.

15. Select Authorization Number Entry

If you do not have the authorization number, you may proceed without it.

16. Save and Continue Process

Click "Save and Continue" to save entered information and proceed.

17. Access Doctor Selection

Click here to open the doctor selection menu and choose your patient's admitting doctor.

18. Select Radiologist

19. Open Additional Doctor Options

Click here to access options for adding or managing referring doctors.

20. Add New Doctor

Click "Add New Doctor" to include a new doctor to the case.

21. Cancel Doctor Addition

22. Save Doctor Selection

Click "Save and Continue" to confirm doctor assignments and proceed.

23. Open Position Selection

Click "Position" to designate the primary diagnosis position of ICD 10 Codes

24. Select Primary Position

25. Access Code Selection

Click "Code" to select the appropriate medical code for the case.

26. Choose Medical Diagnosis Code

27. Open Code Entry Field

28. Enter Medical Code

Enter the medical code number in the field provided.

29. Access Procedure Description

Click "surgical procedure description

30. Enter Procedure Description

Enter a brief description of the surgical procedure, including site and side if applicable.

31. Save Procedure Details

Click "Save and Continue" to save the procedure description and proceed.

32. Access Search Function

complete all required fields in the medical insurance information section.

33. Initiate Search

34. Enter Search Term

35. Select Medical Scheme

36. Access Estimate Amount Field

For Private Patients Click "Enter estimate amount" to input the estimated cost for the procedure.

37. Enter Estimate Amount

Enter the estimated amount for the procedure in the field provided.

38. Save Estimate and Continue

Click "Save and Continue" to save the estimate and proceed to the next step.

39. Access Booking Options

Click here to open booking options for the patient episode.

40. Select Manual Approved Booking

Select the allocated slot for the doctor, or click "Manual Approved Booking" to opt for manual approval of the booking.

41. Open Date Selection

Click here to access the date selection calendar.

42. Choose Appointment Date

43. Confirm Date Selection

44. Access Estimated Minutes Field

Click "Enter Estimated Minutes" to specify the expected duration of the procedure.

45. Enter Procedure Duration

46. Access Time Entry Field

input the scheduled start time for the procedure.

47. Enter Scheduled Time

48. Proceed After Time Entry

Click "Continue" to move forward after entering the scheduled time.

49. Advance to Next Step

upload the surgery consent document if you have it; otherwise, proceed without it.

50. Save Patient Case

Click "Save Case" to finalize and save all entered patient episode details.

51. Confirm Save Action

You have successfully scheduled a new patient episode in Medworkflow by entering patient details, selecting medical options, and finalizing the case. You can now proceed to manage or review scheduled episodes as needed.
